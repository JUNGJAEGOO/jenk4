## Upcoming Products > SimpleData > Overview 

> ※ 본 문서는 alpha 개발 단계의 문서입니다.
> 사용에 관심이 있으신 분은 **support@cloud.toast.com**으로 문의해 주시기 바랍니다.

Toast Cloud SimpleData는 HTTP Restful API를 통하여 간단한 형식의 데이터를 Read/Write 할 수 있는 저장소를 제공하는 서비스입니다.

## 특징

- 데이터의 Sharding을 고민할 필요가 없습니다.
- 객체지향적인 데이터 모델링을 제공합니다.
- Restful API를 통해 손쉽게 데이터에 접근할 수 있습니다.

## 기능

- Class, Object, Property 형태의 객체지향적인 데이터 모델링을 제공합니다.
- String 형식의 Data CRUD 제공.
- HTTP Restful API 제공.

## 서비스 용어

|용어|	설명|
|---|---|
|Property|	Key-Value 로 이루어진 데이터 쌍.|
|Property Key|	사용자가 정의하는 Property 의 이름.|
|Property Value|	사용자가 Property 에 입력한 데이터.|
|Object|	실제 데이터인 Property 들의 모음. 하나의 Class 를 바탕으로 생성할 수 있으며, Property의 Value 를 조작.|
|Class|	Object의 생성을 위한 형식 템플릿. 각 Class 마다 독립적인 Property의 Key를 설정.|
